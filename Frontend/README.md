## üöÄ Quick Start

1. **Clone the repo**  
   ```bash
   git clone https://github.com/flatlogic/angular-material-admin-full.git
   cd angular-material-admin-full
   ```
2. **Install dependencies**  
   ```bash
   npm install
   ```
3. **Run the app**  
   ```bash
   npm start
   ```
   Navigate to http://localhost:3000/.

4. **Build for production**  
   ```bash
   npm build
   ```
