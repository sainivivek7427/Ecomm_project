Requirements:
  • Mac OS X, Windows, or Linux
  • Node.js v6.5 or newer
  • Angular 9

Quick Start:
  • Run npm install
  • Run npm start

There are also other npm tasks:
  • npm build : if you need just to build the app
  • npm lint : to check the source code for syntax errors and potential issues

For more instruction please refer to Angular Material Admin Full readme.md.
