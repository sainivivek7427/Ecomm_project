Angular Material Admin

Angular Material Admin Full is an admin dashboard template built with Angular 9. You can use Angular Material Admin Full to build any type of web applications like SAAS, CMS, financial dashboards, project management tools, etc.

Features
  • Hundreds of Pages
  • Fully Responsive
  • Angular 9
  • Chart Library
  • Theme Change Support
  • E-Commerce Section
  • 2 State Sidebar
  • Fully Documented Codebase
  • And even more coming soon!
